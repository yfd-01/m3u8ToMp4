# m3u8ToMp4
download video by m3u8 url


## Basic Usage:

```python
import m3u8ToMp4

m3u8ToMp4.download("https://demo.com/demo/demo.m3u8", "C:\videos", "video_name")
```

**OR**

```python
import m3u8ToMp4

m3u8ToMp4.download_console()
#Then type params by console
```
